## 🔋 PCB Design Knowledge

I have hands-on knowledge of **PCB designing**, including:

* Schematic design and circuit connectivity
* Component selection based on datasheets
* Footprint assignment and footprint verification
* PCB layout planning
* Track routing and board organization
* Power and ground routing basics
* Design Rule Check, ERC and DRC error fixing
* Gerber file generation
* Basic understanding of manufacturing-ready PCB design

I use PCB design tools to convert circuit ideas into practical and organized hardware layouts.

```text
Circuit Idea → Schematic → Footprint Selection → PCB Layout → DRC Check → Gerber Files → Hardware
```
